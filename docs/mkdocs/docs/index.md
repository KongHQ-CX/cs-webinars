# Kong CS Webinars

Thanks for joining one of Kong's Customer Success-led webinars. This is where we post the slides, sample code, and demo assets from each session.

[Browse webinars](webinars/index.md){ .md-button .md-button--primary }

## Finding your session

Each webinar has its own folder in the repo, named by date and topic, like `2026-07-webinar-name/`. Open the folder for the session you attended and grab what you need: slides, sample code, scripts, a link to the recording.
