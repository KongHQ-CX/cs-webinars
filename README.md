# CS Webinars

This is where Kong's Customer Success team posts the assets from our webinars: slides, sample code, demo scripts, whatever we used on screen.

## Finding a webinar

Start at the [webinar site](https://konghq-cx.github.io/cs-webinars/webinars/): it lists every session with a summary and a one-click download. Prefer the raw files? Browse [`webinars/`](webinars/) in this repo, or use GitHub's search if you already know the topic or date.

## What's in here

Each webinar gets its own folder under [`webinars/`](webinars/), named by date and topic, like `webinars/2026-07-webinar-name/`. Inside is whatever the session used: slides, sample code, scripts, a link to the recording.
