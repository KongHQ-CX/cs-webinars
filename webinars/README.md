# Webinars

Assets from Kong Customer Success-led webinars live here, one folder per session.

Keep individual files under GitHub's 100 MB limit. Don't commit recordings: link to wherever they're hosted instead.

Merging a new folder to `main` automatically zips it and publishes a GitHub Release. See [docs/adding-webinar-assets.md](../docs/adding-webinar-assets.md) for the full process.
